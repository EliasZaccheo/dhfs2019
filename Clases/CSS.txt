Vinculacion interna:
	<style> texto css </style>

Vinculación por línea:
	<p style="texto css"></p>  (el código css se escribe dentro de la etiqueta)

Vinculación externa
	<head>
		<link href="URL o PATH" rel="stylesheet">
	</head> 


p{}	
	Afecta a todas las etiquetas 'p'  (en html: <p>algo</p>)

#id1	
	Afecta al identificador 'id'  (en html: id="id1")

.clase1	
	Afecta a la clase 'clase' (en html class="clase1 clase2")

h1.azul
	Afecta a las etiquetas 'h1' que sean de la clase 'azul'

body p
	Afecta a las etiquetas 'p' contenidas dentro de las etiquetas 'body'

ul#lista1 li
	Afecta a las etiquetas 'li' dentro de la lista desordenada (ul) de identificador lista1 (id)
